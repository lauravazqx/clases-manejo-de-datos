# Entorno de Desarrollo para Manejo de Datos

Este repositorio contiene la configuración necesaria para iniciar un entorno de 
desarrollo en **GitHub Codespaces**, diseñado específicamente para la materia de 
**Manejo de Datos**.

## Descripción

El objetivo de este codespace es proporcionar un ambiente listo para usar con 
todas las herramientas y bibliotecas necesarias preinstaladas, permitiendo 
enfocarse directamente en el análisis y manipulación de datos.

## Bibliotecas Incluidas

El entorno instala automáticamente las dependencias listadas en el archivo 
`requirements.txt`. Las principales bibliotecas disponibles son:

*   **pandas**: Para manipulación y análisis de datos estructurados.
*   **matplotlib**: Para la creación de visualizaciones y gráficos estáticos, 
    animados e interactivos.
*   **openpyxl** y **xlrd**: Para la lectura y escritura de archivos Excel 
    (.xlsx, .xls).

## Cómo usar este repositorio

1.  Haz clic en el botón **Code** en la página principal del repositorio.
2.  Selecciona la pestaña **Codespaces**.
3.  Haz clic en **Create codespace on main**.
4.  Una vez iniciado, el entorno ejecutará automáticamente la instalación de los 
    requisitos y estará listo para trabajar.
